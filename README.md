# react-native-edge-to-edge

React-native - apply 'Edge-to-edge' mode for android platform

## Installation

```sh
npm install react-native-edge-to-edge
```

## Usage


```js
import { multiply } from 'react-native-edge-to-edge';

// ...

const result = await multiply(3, 7);
```
